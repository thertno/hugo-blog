真正提升创作效率的方式，不是让 AI 帮你写一篇文章，而是把 AI 接进你的创作流程里：选题、资料、写作、修改、配图、分发，每一步都让它干具体的活。

最近整理了一批适合中文内容创作者使用的 Codex Skills，按用途分类如下。

## 长文写作与资料研究

- **content-research-writer** — 串联选题、资料、提纲、引用和初稿，适合公众号、Newsletter、博客和 X 长文。适合需要完整写作流程的人。
- **NotebookLM Claude Code Skill** — 基于已有资料库写作，适合行业研究、课程稿、深度文章和长文整理，优点是减少胡编乱造。
- **khazix-skills** — 适合公众号长文、万字研究报告、AI 热点分析、深度拆解和观点型长文。

## 去 AI 味与内容诊断

- **Humanizer-zh** — 中文写作者很实用的去 AI 味工具。不是简单改几个词，而是会帮你处理空话、套话、三段式、过度排比这些问题，让稿子更像真人写出来的。更像一个内容诊断工具。
- **dbskill** — 偏内容诊断，适合选题判断、商业表达、hook、小红书标题、爆款拆解，面向中文自媒体和知识型创作者。

## 配图与视觉化

- **ian-xiaohei-illustrations** — 把文章里的观点、流程、情绪和隐喻，转成"小黑"风格正文插图。它不是随便生成一张图，而是真正理解文章内容后的视觉化表达。
- **html-anything** — 封面图、信息图、结构图、图解都能做，适合给文章做包装，也适合把复杂观点视觉化。适合做杂志风文章、知识卡。

## PPT 与社交卡片

- **guizang-ppt-skill** — 把文章观点转成 PPT、演讲图、公众号头图、小红书封面，适合二次分发。
- **guizang-social-card-skill** — 把长文拆成小红书图文和公众号封面，适合分发包装。一篇长文拆成多张卡片，做成可以直接分发的视觉内容，会省很多时间。

## 推荐入门组合

如果你只想先装一小套，建议从这 5 个开始：

1. **Humanizer-zh** — 文字质量
2. **dbskill** — 选题诊断
3. **content-research-writer** — 写作流程
4. **ian-xiaohei-illustrations** — 正文配图
5. **guizang-social-card-skill** — 分发包装

前 3 个解决文字质量，后 2 个解决配图和分发包装。

## 建议

建议先收藏，不用一次装完，按自己的创作需求慢慢补。每个 Skill 都有对应的 GitHub 仓库，搜索名称即可找到。
